# Administration for Native Americans (ANA) (administration-for-native-americans-ana-)
ANA supports Native American communities by providing financial assistance and capacity building, gathering and sharing data, and advocating for improved policies that affect Native Americans, Alaska Natives, and Native Hawaiians.
